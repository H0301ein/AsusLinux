# AsusLinux

Simple CLI for manage Asus laptop on Linux land :)

## Install 

```sh
$ git clone https://github.com/H0301ein/AsusLinux.git
$ cd AsusLinux
$ chmod +x asus.sh
```

### Options:

#### Laptop Status

```sh
$ ./asus.sh info | bat
```

#### Battery limit

```sh
$ ./asus.sh limit 60 | 80
```

#### CPU Profile

```sh
$ ./asus.sh power | balance | performance
```

## Disclaimer!

**Test** in Asus ZenBook UX425EA
